# dotfiles

Just my dotfiles.

## Installation

### macOS

```shell
git clone https://github.com/mrmlnc/dotfiles ~/.dotfiles
cd ~/.dotfiles/macos
bash setup.sh
```

### Windows

```powershell
Set-ExecutionPolicy -Scope Process -ExecutionPolicy Bypass

# Use git or download button on GitHub
git clone https://github.com/mrmlnc/dotfiles ~\.dotfiles
cd ~\.dotfiles\windows
setup.ps1
```
